# tent.css
tent.cssは、クラス名の指定を必要としないCSSフレームワークです。

[詳細](https://mf2fox.github.io/tent-css/)

## Quick start
```html
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/mf2fox/tent-css/tent.min.css">
    <title>Hello</title>
</head>
<body>
    <h1>Hello world!</h1>
</body>
</html>
```
