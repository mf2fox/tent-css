📝About

tent.cssは、クラス名の指定を必要としないCSSフレームワークです。デモはElementsで見ることができます。

このサイトはtent.cssによってスタイリングされています。

✨Features

タグ構造依存無し

特定のタグの中でスタイルが変化する要素を持ちません。そのため、フレームワークの利用方法を学習する必要がありません。

超低容量

tent.cssの容量はv0.0.1時点で1.06kbです。

💡Use case

  • シンプルなブログ
  • 自己紹介サイト

🎓How to use

tent.cssを利用するいくつかの方法を紹介します。

Pure html

手書きのHTMLで利用する場合は、metaタグの中に以下のタグを追加します。

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/mf2fox/tent-css/tent.min.css">

Hello world

<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/mf2fox/tent-css/tent.min.css">
    <title>Document</title>
</head>
<body>
    <h1>Hello world!</h1>
</body>
</html>
